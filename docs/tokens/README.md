# Tokens

Tokens define reusable design values such as spacing, typography, color, and radius.

## Why tokens matter

Tokens help you:
- keep design decisions consistent
- reuse values across components and themes
- separate system values from implementation details

## Recommended approach

Keep tokens framework-agnostic and stable across themes where possible.
