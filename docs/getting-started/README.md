# Getting Started

This section is for package users.

Use it to understand the core concepts, install the package, and ship your first build quickly.

## Quick start for npm users

```bash
npm install rarog
````

```ts
import { build } from 'rarog'

await build({
  entry: './src/index.ts',
})
```
