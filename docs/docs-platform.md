# Платформа документации v2

Платформа документации v2 делает документацию частью DX-стратегии Рарог, а не просто набором markdown-файлов.

## Опорные принципы

1. **Поисковость** — GitBook search + справочный-страницы для вспомогательные классы, Компонентs, themes и migration.
2. **Живое понимание** — интерактивный испытательная площадка и Примеры для прямой вставки.
3. **Системное мышление** — устройство-разборы компонентов, a11y notes, обозревателем токенов и тем.
4. **Уверенность в миграции** — пошаговые guides для перехода с Tailwind, Bootstrap и старых Рарог интерфейс.

## Модель контента

- `docs/SUMMARY.md` — canonical navigation tree
- `.gitbook.yaml` — корень и структура GitBook
- `examples/playground/` — живая demo-поверхность
- `examples/starters/` — каталог официальных стартеров
- `tests/starters/smoke.mjs` — smoke базовый уровень для набора стартеров

## Конкурентные преимущества

- Документация привязаны к реальным package Поверхностьs, а не к абстрактным примерам;
- компоненты документируются вместе с устройство и accessibility-поведением;
- токены и темы объясняются как среда выполнения system, а не только как CSS variables;
- стартеры документируются рядом с install flow и smoke coverage.
