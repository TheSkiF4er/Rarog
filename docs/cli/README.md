# CLI

Use the CLI for local development, builds, and diagnostics.

## Common commands

```bash
rarog build
rarog dev
rarog inspect
````

## When to use the CLI

Use the CLI when you want to:

* run local development flows
* build production output
* inspect config or runtime behavior
