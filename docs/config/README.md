# Config

Configure Rarog with a project config file.

## Example

```ts
import { defineConfig } from 'rarog'

export default defineConfig({
  themes: [],
  plugins: [],
})
````

## What belongs in config

Use config to define:

* themes
* plugins
* build behavior
* runtime options
