# Базовый уровень доступности

Рарог теперь рассматривает доступность как обязательный базовый уровень выпуска, а не как позднее улучшение.

## Базовые правила

- взаимодействие с первичным управлением с клавиатуры для всех интерактивных компонентов;
- ARIA-паттерны только там, где нативной семантики недостаточно;
- общий видимый контур фокуса через `:focus-visible`;
- `prefers-reduced-motion` соблюдается во всплывающих слоях, указателях загрузки и переходах;
- цветовые токены и токены поверхностей выбраны с ориентиром на AA-контраст по умолчанию;
- каждый компонент поставляется с перечнем проверки и точкой зрительной регрессии по доступности.

## Автоматические проверки

- покрытие Vitest/jsdom для вкладок, диалогов, семантики полей выбора и взаимодействий с клавиатуры;
- зрительная регрессия Playwright с включённым ограничением анимации;
- дополнение Storybook для проверки доступности доступно для ручной проверки.
