Утилиты

Базовые утилиты Rarog покрывают layout, отступы, цвета, типографику и размеры. Попробуйте классы d-flex, gap-3, mt-4, text-muted и другие.

Grid & Layout

Сетка rarog-grid — это 12-колоночная система в стиле Bootstrap на базе классов .rg-row и .rg-col с responsive-модификаторами.

Основной контент

Занимает 9/12 ширины на lg и выше. Смещения и порядок можно тонко настраивать с помощью .rg-offset-* и .rg-order-*.

Классы .grid-2, .grid-3, .grid-4 всё ещё доступны как простые helper'ы, но считаются legacy. Для новых layout'ов рекомендуется использовать .rg-row/.rg-col.

Responsive

Rarog поддерживает Tailwind-подобные префиксы sm:, md:, lg:, xl:, 2xl: для адаптивной верстки.

States

State-утилиты позволяют быстро задавать стили для состояний :hover и :focus.

Hover

Наведите на кнопку, чтобы увидеть работу hover:bg-primary и hover:text-primary.

Focus
