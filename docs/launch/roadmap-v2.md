# Дорожная карта v2

Рарог 1.0 закрывает основу платформы. Ветка v2 должна усиливать то, что уже даёт преимущество в production.

## 1. Theme operations

- presets для отраслевых под стороннюю марку сценариев;
- порядок работы согласования темы с approval-статусами;
- экспорт тем в форматы для design handoff.

## 2. Компонентs for dense apps

- расширенный набор table/data-entry паттернов;
- admin-heavy Компоновкаs и navigation shells;
- лучшие defaults для compact/dense интерфейсов.

## 3. Система оформления governance

- более строгие compatibility reports;
- token policy linting;
- package-level выпуск scorecards.

## 4. Migration acceleration

- более сильные codemod/mapping инструменты;
- guided adoption для Bootstrap, Tailwind, Chakra UI, MUI и Ant Design;
- migration Панельs для больших репозиториев.

## 5. Proof layer

- pinned сравнительный замер baselines для внешних стеков;
- visual regression snapshots как часть выпуск evidence;
- richer public quality reports.
