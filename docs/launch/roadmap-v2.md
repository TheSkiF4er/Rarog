# Дорожная карта v2

Rarog 1.0 закрывает основу платформы. Ветка v2 должна усиливать то, что уже даёт преимущество в production.

## 1. Theme operations

- presets для отраслевых white-label сценариев;
- workflow согласования темы с approval-статусами;
- экспорт тем в форматы для design handoff.

## 2. Components for dense apps

- расширенный набор table/data-entry паттернов;
- admin-heavy layouts и navigation shells;
- лучшие defaults для compact/dense интерфейсов.

## 3. Design system governance

- более строгие compatibility reports;
- token policy linting;
- package-level release scorecards.

## 4. Migration acceleration

- более сильные codemod/mapping инструменты;
- guided adoption для Bootstrap, Tailwind, Chakra UI, MUI и Ant Design;
- migration dashboards для больших репозиториев.

## 5. Proof layer

- pinned benchmark baselines для внешних стеков;
- visual regression snapshots как часть release evidence;
- richer public quality reports.
