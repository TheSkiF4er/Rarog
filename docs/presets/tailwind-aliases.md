# Псевдонимы вспомогательных классов в духе Tailwind

Используйте этот preset, когда adoption начинается с utility-heavy codebase и нужно сохранить знакомый Tailwind vocabulary.

## Включает

- display aliases (`flex`, `grid`, `hidden`)
- Компоновка вспомогательные средства (`justify-between`, `items-center`)
- radius/shadow aliases
- a11y вспомогательные средства (`sr-only`)
