# Components

Core component pack v1 provides a production-ready minimum set with CSS primitives, a11y hooks, examples, and visual coverage.

## Included in v1

- Button
- Input
- Textarea
- Select
- Checkbox
- Radio
- Switch
- Card
- Alert
- Badge
- Modal / Dialog
- Dropdown
- Tabs
- Accordion
- Tooltip
- Spinner
- Skeleton

## Contracts

- Shared focus ring and reduced motion behavior live in `packages/components/src/_component-pack-v1.css`
- Interactive behaviors are powered by `@rarog/js` for Dropdown, Modal, Tooltip, Select and Tabs
- Visual regression coverage lives in `tests/visual/fixtures/components-v1.html`
