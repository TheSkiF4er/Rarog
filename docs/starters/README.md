# Official starters

Official starters — это production-ориентированные шаблоны для основных стеков и распространённых product shapes.

## Included starters

- `vanilla-app`
- `vite-react`
- `nextjs-rarog`
- `vue-vite`
- `nuxt-rarog`
- `dashboard-starter`
- `saas-starter`
- `white-label-starter`

## Install flow

1. choose a starter from `examples/starters/`
2. copy it into a new workspace
3. install dependencies if the starter includes `package.json`
4. run the provided dev script or open the static entrypoint

## Smoke coverage

Starter fleet is validated by `npm run test:starters` and included in CI smoke.
