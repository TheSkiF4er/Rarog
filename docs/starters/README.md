# Официальные стартеры

Официальные стартеры — это production-ориентированные шаблоны для основных стеков и распространённых product shapes.

## Что включено

- `vanilla-app`
- `vite-react`
- `nextjs-rarog`
- `vue-vite`
- `nuxt-rarog`
- `dashboard-starter`
- `saas-starter`
- `white-label-starter`

## Путь установки

1. выберите стартер из `examples/starters/`;
2. скопируйте его в новый workspace;
3. установите зависимости, если у стартерa есть `package.json`;
4. запустите указанный dev-скрипт или откройте статический entrypoint.

## Smoke-покрытие

Набор стартеров проверяется через `npm run test:starters` и включён в CI smoke.
