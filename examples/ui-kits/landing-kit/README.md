# Rarog Landing Kit

Набор типовых лендинговых секций, собранных на Rarog:

- hero‑секция с CTA;
- блок преимуществ (features);
- pricing‑секции (мини и полная);
- блог + FAQ;
- финальный CTA‑блок.

Путь: `examples/ui-kits/landing-kit/index.html`.

Подходит для:

- SaaS‑продуктов;
- маркетинговых лендингов;
- промо‑страниц внутри Cajeer‑экосистемы.
