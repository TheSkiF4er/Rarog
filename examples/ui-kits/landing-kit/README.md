# Набор посадочных страниц «Рарог»

Набор типовых лендинговых секций, собранных на Рарог:

- hero‑секция с CTA;
- блок преимуществ (возможности);
- тарифы‑секции (мини и полная);
- блог + Частые вопросы;
- финальный CTA‑блок.

Путь: `examples/ui-kits/landing-kit/index.html`.

Подходит для:

- SaaS‑продуктов;
- маркетинговых лендингов;
- промо‑страниц внутри Cajeer‑экосистемы.
