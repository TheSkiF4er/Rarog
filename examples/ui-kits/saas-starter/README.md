# Rarog SaaS Starter

Мини‑проект, показывающий типичную структуру SaaS‑приложения:

- `auth.html` — layout для входа/регистрации;
- `dashboard.html` — главная панель с блоком usage и активностью;
- `settings.html` — настройки профиля, безопасности и Billing & Usage.

Путь: `examples/ui-kits/saas-starter`.

Идея: взять эти шаблоны как основу и интегрировать с backend/SPA‑логикой:

- Laravel/Blade — см. Laravel Guide;
- React/Next.js — см. соответствующие guides;
- Cajeer‑стек — см. Cajeer Stack Guide.
