# Административный набор «Рарог» — демонстрационная панель

Этот набор показывает пример административной панели, собранной **только на Рарог**:

- боковая колонка + верхняя панель;
- карточки ключевых показателей;
- таблица с панелью инструментов и пагинацией;
- настройки проекта, активность;
- модальное окно и всплывающее сообщение.

Путь: `examples/ui-kits/admin-dashboard/index.html`.

Подходит как отправная точка для:

- внутренних админок;
- панелей управления SaaS‑продуктов;
- Cajeer‑стека (CMS/панели).
