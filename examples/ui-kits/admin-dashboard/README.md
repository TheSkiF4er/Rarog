# Rarog UI Admin – Demo Dashboard

Этот kit показывает пример админ‑панели, собранной **только на Rarog**:

- sidebar + navbar;
- KPI‑карточки;
- таблица с toolbar‑ом и пагинацией;
- настройки проекта, активность;
- модалка и toast.

Путь: `examples/ui-kits/admin-dashboard/index.html`.

Подходит как отправная точка для:

- внутренних админок;
- панелей управления SaaS‑продуктов;
- Cajeer‑стека (CMS/панели).
