# Rarog HTML Starter

Минимальный пример проекта на чистом HTML с подключением Rarog.

## Использование

1. Соберите CSS и JS фреймворка:
   ```bash
   npm install
   npm run build
   ```
2. Откройте `index.html` в браузере (из директории `examples/starters/html-basic`).

Вы можете копировать эту структуру в любой статический проект.
