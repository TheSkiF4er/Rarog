# Официальные стартеры

Эта директория содержит официальный набор стартеров для Рарог.

## Что включено

- vanilla-app
- vite-react
- nextjs-Рарог
- vue-vite
- nuxt-Рарог
- Панель-стартовый шаблон
- saas-стартовый шаблон
- под стороннюю марку-стартовый шаблон

## Валидация

Запуск:

```bash
npm run test:starters
```
