# Rarog + SvelteKit Starter

Пример использования Rarog в SvelteKit‑проекте.

- CSS Rarog подключается глобально в `+layout.svelte`.
- Компоненты используются через HTML‑разметку и data‑атрибуты.
- JS‑ядро может быть подключено через ESM‑импорт или UMD‑скрипт в `app.html` (см. docs).

Этот starter демонстрирует, что Rarog не привязан к конкретному фреймворку и
работает как drop‑in CSS/JS‑слой.
