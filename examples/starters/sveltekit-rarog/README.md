# Рарог + SvelteKit Стартовый шаблон

Пример использования Рарог в SvelteKit‑проекте.

- CSS Рарог подключается глобально в `+layout.svelte`.
- Компоненты используются через HTML‑разметку и data‑атрибуты.
- JS‑ядро может быть подключено через ESM‑импорт или UMD‑скрипт в `app.html` (см. Документация).

Этот стартовый шаблон демонстрирует, что Рарог не привязан к конкретному фреймворку и
работает как drop‑in CSS/JS‑слой.
