# Utilities (overview)

Rarog ships a Tailwind-like utility layer that covers:

- **Layout**: flex, grid, gap, alignment helpers.
- **Spacing**: margin/padding scales (`m-*`, `p-*`, `mt-*`, etc.).
- **Sizing**: width/height (`w-*`, `h-*`, `min-w-*`, `max-h-*`).
- **Typography**: `text-*`, `font-*`, `leading-*`.
- **Effects**: shadows, radius, filters, backdrop filters, blend modes.
- **Scroll**: overflow, overscroll, scroll snap helpers.
- **Print & RTL**: `print:*` utilities, logical spacing (`ms-*`, `me-*`, `ps-*`, `pe-*`).

Most utilities support:

- responsive prefixes: `sm:`, `md:`, `lg:`, `xl:`, `2xl:`;
- state variants: `hover:`, `focus:`, and advanced ones via Variants & JIT.

See the Russian documentation for the full reference, or use the autocomplete
from the VS Code extension to discover class names.
