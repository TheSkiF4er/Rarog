# Integration Guides (overview)

High-level view of how Rarog integrates with different stacks:

- Laravel (Blade templates + JIT build).
- React (Vite + Rarog).
- Vue.
- Next.js.
- Cajeer stack.

Each stack has its own dedicated guide:

- `guide-laravel`,
- `guide-react`,
- `guide-vue`,
- `guide-nextjs`,
- `guide-cajeer-stack`.
