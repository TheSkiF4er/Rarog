# Variants & JIT

Explains:

- which variants Rarog supports (`hover:`, `focus:`, `group-hover:`,
  `peer-*`, `data-[state=*]`, responsive prefixes, etc.);
- how the JIT engine scans your content (`content` globs in `rarog.config.*`);
- how to use arbitrary values (`w-[320px]`, `bg-[#0f172a]`, ...);
- how `mode: "full"` vs `mode: "jit"` affects your bundle size.
