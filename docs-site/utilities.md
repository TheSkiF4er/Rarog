# Utilities

Утилиты Rarog покрывают:

- Display и layout:
  - `d-block`, `d-none`, `d-flex`, `d-grid`, responsive-варианты `sm:d-flex` и т.п.
- Flex/Grid-направления:
  - `flex-row`, `flex-column`, `justify-*`, `items-*`, `gap-*`.
- Spacing:
  - `mt-1..4`, `mb-1..4`, `p-2..4` и responsive-варианты `sm:mt-4`.
- Typography:
  - `text-left`, `text-center`, `text-right`, `text-muted`.
- Visibility & accessibility:
  - `d-none`, `sr-only`, overflow-утилиты.

Подробнее см. исходник:

```text
packages/utilities/src/rarog-utilities.css
```
