# Материалы системы оформления «Рарог»

Каталог `design/` содержит артефакты дизайн‑системы Рарог, которые могут быть
использованы в Figma и других дизайн‑инструментах.

- `figma.tokens.json` — экспорт дизайн‑токенов в формате, совместимом с Tokens Studio.
- `figma-kit/` — описание Figma Design Kit (структура, компоненты, Компоновка‑паттерны).
