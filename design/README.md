# Rarog Design System Assets

Каталог `design/` содержит артефакты дизайн‑системы Rarog, которые могут быть
использованы в Figma и других дизайн‑инструментах.

- `figma.tokens.json` — экспорт дизайн‑токенов в формате, совместимом с Tokens Studio.
- `figma-kit/` — описание Figma Design Kit (структура, компоненты, layout‑паттерны).
