## [1.0.0] – 2025-12-12

Первый стабильный релиз **Rarog CSS**.

### Добавлено

#### Ядро (Rarog Core)

- Введена базовая система **дизайн-токенов** на основе CSS-переменных:
  - Цветовые токены:
    - Брендовые (`--rarog-color-primary`, `--rarog-color-secondary` и их foreground-варианты).
    - Семантические (`--rarog-color-success`, `--rarog-color-danger`, `--rarog-color-warning`, `--rarog-color-info`).
    - Фоновые и текстовые (`--rarog-color-bg`, `--rarog-color-bg-soft`, `--rarog-color-surface`, `--rarog-color-border`, `--rarog-color-text`, `--rarog-color-text-muted`).
  - Токены отступов (`--rarog-space-*`) с шагами от `0` до `12`.
  - Токены скруглений (`--rarog-radius-xs` … `--rarog-radius-full`).
  - Токены теней (`--rarog-shadow-xs` … `--rarog-shadow-lg`).
- Базовый `reset` и нормализация стилей:
  - Сброс `box-sizing` для всех элементов.
  - Базовая типографика и поведение `html`, `body`, форм и медиа-элементов.
- Глобальные стили:
  - Гладкий скролл (`scroll-behavior: smooth`).
  - Минимальная высота для `body` (`min-height: 100vh`).
- Базовый контейнер:
  - Класс `.rg-container` с ограничением ширины, выравниванием по центру и базовыми внутренними отступами.

#### Utility-слой (Rarog Utilities)

- **Layout и Flex/Grid-утилиты**:
  - Отображение: `.d-block`, `.d-inline`, `.d-inline-block`, `.d-flex`, `.d-inline-flex`, `.d-grid`.
  - Направление потока: `.flex-row`, `.flex-column`.
  - Перенос: `.flex-wrap`, `.flex-nowrap`.
  - Выравнивание по основной оси: `.justify-start`, `.justify-center`, `.justify-between`, `.justify-end`.
  - Выравнивание по поперечной оси: `.items-start`, `.items-center`, `.items-end`.
  - Управление промежутками: `.gap-1`, `.gap-2`, `.gap-3`, `.gap-4`, `.gap-6`.
  - Простые шаблоны сетки: `.grid-2`, `.grid-3`, `.grid-4`.
- **Утилиты отступов (Spacing)**:
  - Маргины: `.m-0`, `mt-1`, `mt-2`, `mt-3`, `mt-4`, `mb-2`, `mb-3`, `mb-4`.
  - Паддинги: `.p-0`, `.p-2`, `.p-3`, `.p-4`, `.p-6`.
- **Цвет и границы**:
  - Фоновые цвета: `.bg-primary`, `.bg-secondary`, `.bg-surface`.
  - Цвет текста: `.text-primary`, `.text-muted`.
  - Базовая рамка: `.border` с использованием токена `--rarog-color-border`.
  - Вариант рамки по брендовому цвету: `.border-primary`.
  - Скругления: `.rounded-md`, `.rounded-lg`.
- **Типографика**:
  - Размеры текста: `.text-xs`, `.text-sm`, `.text-base`, `.text-lg`, `.text-xl`.
  - Жирность: `.font-semibold`, `.font-bold`.
  - Выравнивание текста: `.text-left`, `.text-center`, `.text-right`.
- **Размеры блоков**:
  - Ширина/высота: `.w-full`, `.h-full`.
  - Максимальная ширина: `.max-w-sm`, `.max-w-md`, `.max-w-lg`.

#### Компонентный слой (Rarog Components)

- **Кнопки**:
  - Базовый класс `.btn`:
    - Выравнивание по центру, inline-flex, управление `gap`.
    - Паддинги и скругления через токены.
    - Анимации наведения и нажатия (transition + лёгкое смещение по оси Y).
  - Варианты:
    - `.btn-primary` — основная кнопка на брендовом цвете.
    - `.btn-secondary` — вторичная кнопка.
    - `.btn-outline` — прозрачный фон с рамкой.
  - Размеры:
    - `.btn-sm` — компактная кнопка.
    - `.btn-lg` — крупная кнопка.
  - Состояния:
    - `:disabled` — визуальное состояние отключённой кнопки.
    - `:hover` для основного варианта с усилением тени и небольшим подъёмом.
- **Карточки**:
  - `.card` — контейнер с фоном, рамкой, скруглением и лёгкой тенью.
  - `.card-header` — заголовок карточки с отступами и усиленной жирностью.
  - `.card-footer` — нижняя область карточки для вспомогательного текста или действий.
- **Формы**:
  - `.field` — обёртка для группы `label + input`.
  - `.label` — подпись к полю ввода с уменьшенным размером и muted-цветом.
  - `.input` — текстовое поле с:
    - шириной 100%;
    - паддингами и скруглением через токены;
    - состоянием `:focus` (подсветка рамки и псевдо-outline).
- **Навигация (Navbar)**:
  - `.navbar` — шапка сайта с фоном, рамкой снизу и вертикальными паддингами.
  - `.navbar-inner` — контейнер для контента navbar (бренд + навигация) с `flex`-разметкой.
  - `.navbar-brand` — бренд/логотип, выделенный усиленной типографикой.
  - `.navbar-nav` — контейнер для ссылок навигации.
  - `.navbar-link` — стили для ссылок навигации и их hover-состояния.

#### Темы (Rarog Themes)

- **Светлая тема** (`rarog-theme-default.css`):
  - Значения фоновых и текстовых токенов, адаптированных под светлый фон:
    - Светлый фон (`--rarog-color-bg`, `--rarog-color-bg-soft`, `--rarog-color-surface`).
    - Светлые границы (`--rarog-color-border`).
    - Тёмный основной текст и приглушённый вторичный.
- **Тёмная тема** (`rarog-theme-dark.css`):
  - Определения внутри селектора `:root.theme-dark`:
    - Тёмные фоны и поверхности.
    - Контрастный светлый текст и muted-вариант.
  - Поддержка переключения тем через класс `.theme-dark` на `html` или `body` без изменения HTML-разметки компонентов.

#### Документация и демо

- Базовая демо-страница `docs/index.html` (на русском языке):
  - Подключение `Rarog Core`, `Utilities`, `Components` и `Dark Theme`.
  - Navbar с навигацией по секциям документации.
  - Пример карточки, кнопки и текстовых блоков.
  - Стартовая секция для демонстрации utility-классов.

#### Инфраструктура и инструменты

- Структура монорепозитория `packages/*`:
  - `packages/core` — ядро и токены.
  - `packages/utilities` — утилиты.
  - `packages/components` — компоненты.
  - `packages/themes` — темы.
- Конфигурационный файл `rarog.config.json`:
  - Базовая информация о версии.
  - Ссылки на файлы токенов.
  - Настройки сборки по пакетам и темам.
- `package.json`:
  - Метаданные пакета (`name: "rarog-css"`, версия `1.0.0`, описание, автор, лицензия).
  - Скрипты:
    - `npm run build` — сборка всех пакетов.
    - `npm run build:core`, `build:utilities`, `build:components`, `build:themes` — таргетированные сборки.
    - `npm run docs` — запуск dev-сервера документации.
- Скрипты в `tools/`:
  - `build.mjs` — базовый сборщик CSS:
    - Копирование исходных файлов из `src` в `dist` (заготовка под интеграцию PostCSS/Autoprefixer/CSSNano).
  - `docs-dev.mjs` — простой HTTP-сервер для локального поднятия документации на `http://localhost:4173`.
- GitHub-инфраструктура:
  - `.github/ISSUE_TEMPLATE/bug_report.yml` — шаблон для отчёта об ошибке.
  - `.github/ISSUE_TEMPLATE/feature_request.yml` — шаблон для запроса фич.
  - `.github/PULL_REQUEST_TEMPLATE.md` — шаблон Pull Request с чек-листом.
  - `.github/workflows/release.yml` — GitHub Actions workflow:
    - Триггер по пушу тегов `v*.*.*`.
    - Установка Node.js, установка зависимостей и сборка проекта.
    - Публикация артефактов сборки (dist).
