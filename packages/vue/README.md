# @rarog/vue

Согласующий слой Vue v1 for Рарог with Пригодный для Nuxt SSR usage, composable-style вспомогательные средства and wrapped Компонентs for core UI базовые элементы.

Входящие оболочки:
- `RarogProvider`
- `RarogButton`
- `RarogInput`
- `RarogTextarea`
- `RarogSelectField`
- `RarogCheckbox`
- `RarogRadio`
- `RarogSwitch`
- `RarogCard`
- `RarogAlert`
- `RarogBadge`
- `RarogSpinner`
- `RarogSkeleton`
- `RarogTabs`
- `RarogAccordion`
- `RarogTooltip`
- `RarogModal`
- `RarogOffcanvas`
- `RarogDropdown`

Composable вспомогательные средства:
- `useRarogInit`
- `useModal`
- `useOffcanvas`
- `useDropdown`
- `useTooltip`
- `useCollapse`

Примеры:
- `examples/starters/nuxt-rarog`
