# @rarog/vue

Vue adapter v1 for Rarog with Nuxt-friendly SSR usage, composable-style helpers and wrapped components for core UI primitives.

Included wrappers:
- `RarogProvider`
- `RarogButton`
- `RarogInput`
- `RarogTextarea`
- `RarogSelectField`
- `RarogCheckbox`
- `RarogRadio`
- `RarogSwitch`
- `RarogCard`
- `RarogAlert`
- `RarogBadge`
- `RarogSpinner`
- `RarogSkeleton`
- `RarogTabs`
- `RarogAccordion`
- `RarogTooltip`
- `RarogModal`
- `RarogOffcanvas`
- `RarogDropdown`

Composable helpers:
- `useRarogInit`
- `useModal`
- `useOffcanvas`
- `useDropdown`
- `useTooltip`
- `useCollapse`

Examples:
- `examples/starters/nuxt-rarog`
