# @rarog/react

React adapter v1 for Rarog with typed props, SSR-safe primitives, controlled/uncontrolled patterns and wrapped component APIs.

Included wrappers:
- `RarogProvider`
- `RarogButton`
- `RarogInput`
- `RarogTextarea`
- `RarogSelectField`
- `RarogCheckbox`
- `RarogRadio`
- `RarogSwitch`
- `RarogCard`
- `RarogAlert`
- `RarogBadge`
- `RarogSpinner`
- `RarogSkeleton`
- `RarogTabs`
- `RarogAccordion`
- `RarogTooltip`
- `RarogModal`
- `RarogOffcanvas`
- `RarogDropdown`

Hooks:
- `useRarogInit`
- `useModal`
- `useOffcanvas`
- `useDropdown`
- `useTooltip`
- `useCollapse`

Examples:
- `examples/starters/vite-react`
- `examples/starters/nextjs-rarog`
