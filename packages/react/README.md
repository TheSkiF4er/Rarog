# @rarog/react

Согласующий слой React v1 for Рарог with типизированный свойства, Пригодный для серверного вывода базовые элементы, controlled/uncontrolled patterns and wrapped Компонент APIs.

Входящие оболочки:
- `RarogProvider`
- `RarogButton`
- `RarogInput`
- `RarogTextarea`
- `RarogSelectField`
- `RarogCheckbox`
- `RarogRadio`
- `RarogSwitch`
- `RarogCard`
- `RarogAlert`
- `RarogBadge`
- `RarogSpinner`
- `RarogSkeleton`
- `RarogTabs`
- `RarogAccordion`
- `RarogTooltip`
- `RarogModal`
- `RarogOffcanvas`
- `RarogDropdown`

Hooks:
- `useRarogInit`
- `useModal`
- `useOffcanvas`
- `useDropdown`
- `useTooltip`
- `useCollapse`

Примеры:
- `examples/starters/vite-react`
- `examples/starters/nextjs-rarog`
